commands until now:
git init -> initialized git into a file
git status -> changes made or not(we get to know throught this command)
git add -> to make any file "Staged" (ready to perform commit)
git add . ->to add all files in "Staged"
git commit -m "your msg about commit" -> commiting changes(now files are unchanged) -m means message
git push origin main -> upload local content into remote repository(github).
